Crystal.IA

Bot de Telegram con IA multi-modelo mediante OpenRouter API

Simulación de Chat

Usuario: ¿Qué es Crystal.IA?
Crystal.IA: Soy un chatbot con 7 modelos de IA diferentes, incluyendo DeepSeek, Llama y más. Puedo ayudarte con preguntas, creatividad y análisis.
Usuario: ¿Cómo funciona tu arquitectura?
Crystal.IA: Uso OpenRouter API para acceder a múltiples modelos, tengo un panel web Flask para monitoreo, base de datos JSON con backup automático y sistema de rate limiting.

🏗️ Arquitectura Principal

Sistema bot Telegram con IA multi-modelo mediante OpenRouter API. Arquitectura en capas:

  • Interfaz Telegram: Comunicación con usuarios
  • Lógica de negocio: Procesamiento de comandos
  • Integración IA: Conexión con OpenRouter API
  • Persistencia JSON: Base de datos local
  • Panel web Flask: Monitoreo en tiempo real

Search Nexus

Bot de búsqueda de multimedia en Telegram con imágenes y 5 categorías

Simulación de Búsqueda

Usuario: /search
Search Nexus: Selecciona categoría:
🎬 Películas | 📺 Series | 🇯🇵 Anime
🎮 Videojuegos | 🎨 Animadas
Usuario: 🎬 Películas
Search Nexus: ¿Qué película buscas?
Usuario: Avengers Endgame
Search Nexus: ✅ Encontrado: Avengers: Endgame
⭐ 8.4/10 | 🗓️ 2019 | ⏱️ 181min
🎭 Acción, Aventura, Ciencia ficción
📝 Los Vengadores se reúnen para revertir el chasquido de Thanos.

🎯 Objetivo

Bot de Telegram que busca información sobre contenido de entretenimiento con imágenes en 5 categorías diferentes.

🚀 Funcionalidades Principales

5 Categorías

Películas, series, anime, videojuegos y series animadas

Con Imágenes

Muestra pósters y capturas cuando están disponibles

Tendencias

Contenido trending del día en todas las categorías

Aleatorio

Recomendaciones aleatorias para descubrir contenido nuevo

⚡ Comandos Disponibles

/start Menú principal con lista de comandos
/search Buscar contenido (activará botones de categorías)
/tendencias Contenido trending del día
/aleatorio Recomendación aleatoria
/estrenos Próximos estrenos de cine
/juegos Mejores videojuegos por rating
/help Ayuda e instrucciones completas

🔄 Flujo de Búsqueda

  1. Usuario escribe /search
  2. Selecciona categoría con botones interactivos
  3. Escribe el nombre del contenido
  4. Recibe información detallada con imagen

🔧 APIs Utilizadas

  • The Movie Database (TMDB): Para películas y series
  • RAWG: Para videojuegos
  • TMDB + Kitsu: Para anime (solución doble para mayor precisión)

⚙️ Características Técnicas

  • Interfaz basada en botones para navegación
  • Manejo de errores robusto
  • Timeouts configurados
  • Formato Markdown para mejor presentación
  • Descarga y envío de imágenes optimizado
  • Resultados rápidos e intuitivos
